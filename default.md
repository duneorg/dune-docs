A flat-file CMS built on [Deno](https://deno.land/) and [Fresh 2](https://fresh.deno.dev/). Content is files. No database required.

## Choose your path

Dune serves content editors, webmasters, and developers equally. Pick your starting point:

### Content Editor

Write pages in Markdown, organize with folders, tag with taxonomies. No code required.

**Start with:** [Getting Started](getting-started) → [Content](content) → [Reference](reference)

### Webmaster

Configure, deploy, and manage a Dune site. YAML config, CLI tools, multiple environments.

**Start with:** [Getting Started](getting-started) → [Configuration](configuration) → [Deployment](deployment)

### Developer

Build themes, write TSX content pages, create plugins, extend the engine.

**Start with:** [Getting Started](getting-started) → [Content](content) → [Themes](themes) → [Extending](extending)
